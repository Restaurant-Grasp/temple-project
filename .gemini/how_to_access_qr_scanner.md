## ğŸ”§ **Quick Troubleshooting**

### **Issue: Can't find "QR Scanner" in menu**
**Solution:** 
1. Refresh the page (F5 or Ctrl+R)
2. Clear browser cache
3. The menu item was just added - it should appear after refresh

### **Issue: Camera not starting**
**Solution:**
1. Check camera permissions in browser
2. Make sure you're on HTTPS (secure connection)
3. Try using manual verification instead

### **Issue: QR code not detected**
**Solution:**
1. Improve lighting
2. Move camera closer/farther
3. Hold camera steady
4. Use manual verification as backup
